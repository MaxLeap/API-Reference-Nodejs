Class: Relation

Relation

A class that is used to access all of the children of a many-to-many relationship. Each instance of ML.Relation is associated with a particular parent object and key.

Constructor

new Relation(parent, key)

Creates a new Relation for the given parent object and key. This constructor should rarely be used directly, but rather created by ML.Object.relation.
Parameters:
Name Type Description
parent ML.Object The parent of this relation.
key String The key for this relation on the parent.
